# KWEngine
---
## 개요
해당 프로젝트는 크래프톤 정글 게임 테크랩 3기를 진행하면서 복습 목적으로 제작하는 자체 제작 엔진 입니다.

## 개발 범위
DirectX11을 사용한 게임 엔진 개발에 필요한 기능들을 위주로 학습합니다.

## 사용 프레임워크
- DirectX11
- C++ 20
- Visual studio 2022
---
## 특징
- 엔진 구조는 레퍼런스 없이 저의 판단 + 제가 수용하는 AI가 추천하는 구조 기반으로 작성됩니다.
- AI를 사용하는 소위 '뇌빼기 코딩' 없이 순수 타자를 사용해서 일일히 구현합니다.
